## 四、API接口总览

### 用户管理 (`/admin/user`)
| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | `/list` | 获取用户列表 |
| GET | `/{id}` | 获取用户详情 |
| PUT | `/{id}/status` | 更新用户状态 |
| GET | `/stats` | 获取用户统计 |

### 梦境管理 (`/admin/dream`)
| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | `/list` | 获取梦境列表 |
| GET | `/{id}` | 获取梦境详情 |
| DELETE | `/{id}` | 软删除梦境 |
| PUT | `/{id}/status` | 更新梦境状态 |
| PUT | `/{id}/privacy` | 更新梦境公开状态 |
| GET | `/stats` | 获取梦境统计 |

### 评论管理 (`/admin/comment`)
| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | `/list` | 获取评论列表 |
| GET | `/{id}` | 获取评论详情 |
| DELETE | `/{id}` | 软删除评论 |
| PUT | `/{id}/restore` | 恢复评论 |
| GET | `/stats` | 获取评论统计 |

### 仪表盘 (`/admin/dashboard`)
| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | `/stats` | 获取仪表盘统计数据 |

### 管理员管理 (`/admin/admin`)
| 方法 | 路径 | 描述 |
|-----|------|------|
| GET | `/list` | 获取管理员列表 |
| GET | `/{id}` | 获取管理员详情 |
| POST | `/create` | 创建管理员 |
| PUT | `/{id}/status` | 更新管理员状态 |
| DELETE | `/{id}` | 删除管理员 |
| GET | `/stats` | 获取管理员统计 |
